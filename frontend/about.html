About This Project

Game Sales Explorer is a full-stack web application developed to explore and analyze a video game sales dataset.

Technologies Used

  • Python
  • FastAPI
  • Pandas
  • HTML
  • CSS
  • JavaScript

Features

  • Search games
  • Filter by platform and genre
  • Sort by Top Sales, Newest and Oldest
  • Interactive game pages
  • REST API with Swagger documentation
